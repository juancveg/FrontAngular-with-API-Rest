=========================================
Aplicación Web de Posts y Comentarios
=========================================

Ruta de acceso al frontend de Angular:

http://localhost:4200/posts

Pasos para ejecutar la aplicación:

1. Abre una terminal en la carpeta raíz del proyecto.
2. Ejecuta el comando: npm install
3. Luego ejecuta: npm start
4. Abre tu navegador y accede a: http://localhost:4200/posts

Estructura principal:
- Listado de posts al inicio.
- Filtro de posts por usuario.
- Visualización de comentarios de cada post.
